<pre lang="markdown"><code> # 🤖 Telegram Bot: Персонализированный Ассистент Этот бот предоставляет персонализированные рекомендации и поддерживает интерактивное общение с пользователями. ## 📌 Требования и функциональность 1. 🧠 **Адаптивность** Бот анализирует запросы пользователя и выдает персонализированные рекомендации на основе базы данных. 2. 🗨 **Интерактивность** - Поддержка текстовых сообщений - Кнопки и встроенные меню (inline keyboard) 3. 📋 **Удобный формат информации** - Использование emoji, списков и форматирования - Ответы оптимизированы под Telegram UI 4. 🗄 **Работа с базой данных** Бот подключен к SQLite (или другой СУБД), данные можно дополнять вручную или автоматически. ## 📁 Структура проекта telegram-bot/ ├── bot.py # Основная логика бота ├── db.py # Работа с базой данных ├── config.py # Конфигурации и токены ├── requirements.txt # Зависимости └── README.md # Документация ## 🧩 Пример данных в базе Таблица: `users_data` | Поле | Тип | Описание | |-------------|-----------|-----------------------------------| | id | INTEGER | Уникальный ID записи | | user_id | INTEGER | Telegram ID пользователя | | preferences | TEXT | Предпочтения (в формате JSON) | | last_active | DATETIME | Последняя активность пользователя | </code></pre>
